# Cybr lab resources

Public scripts and supporting files for hands-on security labs at
[Cybr](https://cybr.com) and [Cybr Labs](https://labs.cybr.com). 

## Contents

| Path | Used by | What it is |
| --- | --- | --- |
| `wordpress/wp2shell_rce/wp2shell.py` | Hacking WordPress with wp2shell (pre-auth SQLi to RCE)

## A note on usage

These files are built for Cybr's disposable lab environments. Only run them
against systems you own or have explicit permission to test. These are not
meant to be used in any sort of production environment, and theyare not to 
be used for illegal purposes.

Use at your own risk, and don't use them to do stupid things.
We are not responsible for any damages.
